# HTML5

Esse repositório contém diversos pequenos projetos de HTML, CSS e Javascript sugeridos em um blog de desenvolvimento. Você pode ver as idéias dos projetos no link a seguir: https://www.freecodecamp.org/news/javascript-projects-for-beginners/

### Motivação

Embora hoje em dia haja diversas tecnologias diferentes, eficazes e de fácil execução, é muito importante que os desenvolvedores conheçam como manipular o DOM com javascript puro. É fundamental que os desenvolvedores saibam como os frameworks (ReactJS, VueJS e Angular) realizam essas tarefas under the hood.
